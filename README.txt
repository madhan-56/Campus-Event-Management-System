Campus Event Management System - Quick Start
-------------------------------------------
1. Extract the zip.
2. In a terminal, run MongoDB server (e.g., `mongod`) or ensure MongoDB service is running.
3. Open project folder and install dependencies:
   cd campus-event-system
   npm install
4. Start backend:
   node backend/server.js
5. Open frontend files in browser:
   frontend/index.html (for login/register)
   frontend/events.html (events listing)
   frontend/addEvent.html (admin add event)
Notes:
- Default server port: 5000
- API base: http://localhost:5000/api
